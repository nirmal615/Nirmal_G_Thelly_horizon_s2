# Level 1 – Rover Program

## Objective  
This program calculates the distance between two points and the time required for a rover to reach the destination.

## Description  
The user inputs origin and destination coordinates.  
The program calculates distance using the distance formula and computes time using motion equations.

## Output  
Displays:
- Distance  
- Time required
