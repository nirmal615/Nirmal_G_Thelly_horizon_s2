
# Level 3 – ROS2 Task

## Objective

The aim of this task was to understand how ROS2 works and to create a simple communication between two nodes.

## Environment

* Ubuntu 22.04 (installed using VirtualBox)
* ROS2 Humble

## What I Did

First, I installed Ubuntu 22.04 and then set up ROS2 Humble. After confirming that ROS2 was working, I ran the basic demo nodes.

I used:

* **Talker node** as the publisher
* **Listener node** as the subscriber

The talker continuously publishes messages, and the listener receives and prints them in the terminal.

## Output

The publisher was sending messages like:
“Hello World”

The subscriber successfully received those messages and displayed them.

## Proof

Screenshots are included in this folder showing:

* Ubuntu installation
* ROS2 working
* Publisher output
* Subscriber output

## Conclusion

This task helped me understand how ROS2 nodes communicate using a publisher-subscriber model.
